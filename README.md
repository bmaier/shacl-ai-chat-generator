## Kurzanleitung zur Einrichtung

1.  **Platzieren:** Kopieren Sie diesen Ordner nach `_bmad/bsg/`.
2.  **Laden:** Starten Sie Ihre Gemini-CLI neu.
3.  **Setup:** Nutzen Sie den Befehl `/bsg_configure_governance` (Auto-Init).

Detaillierte Anweisungen finden Sie im **[KORRIGIERTEN Installations-Guide](docs/installation.md)**.
