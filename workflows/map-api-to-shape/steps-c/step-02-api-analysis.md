# Step 2: Struktur-Analyse

## STEP GOAL:
Identifikation der Datenmodelle (Schemas) innerhalb der API, die semantisch beschrieben werden sollen.

## MANDATORY SEQUENCE
1. Scanne die Sektion `components/schemas`.
2. Liste alle gefundenen Objekte und deren Eigenschaften auf.
3. Frage den Nutzer, ob alle oder nur bestimmte Objekte gemappt werden sollen.
